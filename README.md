# 🍽 Restaurant POS System

A simple Restaurant Point of Sale (POS) System built with **C++**.

## Features

- Display restaurant menu
- Add/Edit/Remove orders
- Calculate total & discount
- Print invoice
- Save orders as JSON

## Technologies

- C++
- STL
- nlohmann/json

## Project Structure

```text
Restaurant POS System
│
├── data/
├── docs/
├── include/
├── src/
├── main.cpp
└── README.md
```

## Build

```bash
g++ src/*.cpp main.cpp -Iinclude -o restaurant.exe
```
